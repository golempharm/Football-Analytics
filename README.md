# Football Analytics Dashboard

Aplikacja Streamlit do analizy danych meczowych z pliku JSON w formacie StatsBomb.  
Dashboard umożliwia wybór zawodnika i prezentuje podstawowe statystyki, takie jak podania, celność podań, strzały, xG, xA oraz podania progresywne.

Aplikacja zawiera również wizualizacje:
- heatmapę aktywności zawodnika,
- mapę podań,
- mapę strzałów,
- passing network,
- radar statystyk zawodnika.

Dane są wgrywane przez użytkownika jako plik `.json`.
